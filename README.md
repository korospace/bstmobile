# Bank Sampah Teratai [BST]
## Required
<ul>
    <li> NodeJS 16.13.0 </li>
    <li> npm 8.0.0  </li>
</ul>

### Install
<ol>
    <li> Clone
    <li> Change Directory to root project </li>
    <li> run >> npm install </li>
</ol>

### Build
<ol>
    <li> run >> ionic cap build android --prod
</ol>

### Assets Stuff
<ol>
    <li> Add following
